# Installation

View justjake's [readme](https://github.com/justjake/drc-sim-keyboard/blob/master/README.md).

# Controls

The controls are hardcoded right now,  but i hope to move them to a config file.
| Gamepad | Keyboard |
| ------- | -------- |
| Up Down Left Right | Arrow Keys |
| a       | SPACE    |
| b       | e        |
| x       | d        |
| y       | f        |
| l       | q        |
| r       | r        |
| zl      | t        |
| zr      | g        |
| minus   | x        |
| plus    | c        |
| home    | z        |
